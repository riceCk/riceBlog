基本类型

  • number: 数字
  • string: 字符串
  • boolean：布尔
  • 数组
  • object: 对象
  • null 和 undefined

null和undefined是其他类型的子类型，它们可以赋值给其他类型
通过添加strictNullChecks: true,可以获得更严格的空类型检查，null和undefined只能赋值给自身

其他常用类型

  • 联合类型

配合类型保护进行判断
类型保护：当对某个变量进行类型判断之后，在判断的语句中便可以确定它的确切类型，typeof可以触发类型保护

  • void类型:通常用于约束函数的返回值，表示该函数没有任何返回
  • never类型: 通常用来约束函数的返回值，表示该函数永远不可能结束

  • 字面量类型:使用一个值进行约束

  • 元祖类型（Tuple）:一个固定长度数组，并且数组中每一项的类型确定

  • any类型: any类型可以绕过类型检查，因此,any类型的数据可以赋值给任意类型
