# 类型变化之美
## 显示类型转换
 
| 类型运算符       | 注解          | 结果  |
| ------------- |-------------  | ------|
| Number(mix)     |  转换成数字类型 | 数字、NaN |
| parseInt(string,radix)     | 默认转换成十进制，布尔值不算      |   radix为进制数 |
| parseFloat(string) | 转换成小数      |    数字、NaN |
| toString(radix ) | are neat      |    $1 |
| String | are neat      |    $1 |
| Boolean | are neat      |    $1 |
