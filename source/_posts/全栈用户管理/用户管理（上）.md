# 前期配置
## 依赖下载
### 全局安装下载express框架（初始下载就够了）
```bash
npm install express-generator -g
```
### 安装express模板框架
```bash
express -e
```
### 安装none热模块加载（初始下载）
```bash
npm install nodemon -g
```
### 安装mysql
```bash
npm install mysql -s
```
