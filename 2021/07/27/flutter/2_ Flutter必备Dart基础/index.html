1_Flutter必备Dart基础

发表于 2021-07-27 | 分类于 flutter |

Mixins

  • Mixins类似于多继承，是在多类继承中重用一个类代码的方式
  • 作为Mixin的类不能有显示声明构造方法
  • 作为Mixin的类只能继承自Object
  • 使用关键字with连接一个或多个mixin

操作符覆写

  • 腹泻操作符需要在类中定义
  • 如果覆写 ==，还需要覆写对象的hashCodegetter方法
< + I []
> / ^ []=
<= ~/ & ~
>= * << ==
- % >>

枚举

  • 枚举是一种有穷序列集的数据类型
  • 使用关键字enum定义一个枚举
  • 常用于代替常量，控制语句等

Dart枚举特性

  • index从0开始的，依次累加
  • 不能指定原始值
  • 不能添加方法

泛型

  • Dart中类型是可选的，可使用泛型限定类型
